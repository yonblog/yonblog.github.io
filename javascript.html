JavaScript DOMの用語解説

Nodeツリー・DOMツリー


              HTMLの要素は階層構造になっている。

Element Node

              
              HTML,body,h1などの要素こと。

Text Node


              テキストや空白の要素こと。

Parent Node


              親ノードのこと。

Child Node


              子ノードのこと。

Sibling Node(シブリングノード)


              同じ階層の兄弟ノードのこと。

要素名.ChildNodes


              要素名の全ての子ノードを取得できる。

              ※空白・改行も含まれる。

要素名.Children


              空白・改行ノードを除いた、要素名の全ての子ノードを取得できる。

要素名.FirstChild 要素名.LastChild


              子ノードの最初、子ノードの最後が取得できる。

              ※空白が最初なら空白が取得される。改行が最初なら改行が取得される。

要素名.FirstElementChild 要素名.LastElementChild


              子ノードの最初、子ノードの最後が取得できる。

              ※空白・改行は含まれない。

要素名.parentNode


              要素の親ノードが取得できる。

要素名.previousSibling 要素名.nextSibling


              1つ前の兄弟ノード、1つ後の兄弟ノードが取得できる。

              ※空白が最初なら空白が取得される。改行が最初なら改行が取得される。

要素名.previousElementSibling 要素名.nextElementSibling


              1つ前の兄弟ノード、1つ後の兄弟ノードが取得できる。

              ※空白・改行は含まれない。

ドットインストール / JavaScript オブジェクト編

今の日時を取得したいとき

              
              new Date() → インスタンスとして生成すると、現在日時を定数に取得できる。

              getFullYear() → 年を取得できる。

              getMonth() → 月を取得できる。※「1月→ 0」「2月→ 1」「5月→ 4」などとなっているので注意。

              getDate() → 日を取得できる。

              getDay() → 曜日を取得できる。※「0 →日曜」「1 →月曜」「2 →火曜」

              getHours() → 時間を取得できる。

              getMinutes() → 分を取得できる。

              getSeconds() → 秒を取得できる。

              getMilliseconds() → ミリ秒を取得できる。

特定の日時を指定したいとき

              
              setHours() → setHours(10, 20, 30); とすると「10時20分30秒」になる。

              setDate() → setDate(30); とすると「30日」になる。

alert()、confirm()

              
              alert() → 警告を表示させたいときに使う。

              confirm() → 確認を求めるときに使う。(「キャンセルしますか？」など)

setInterval()、clearInterval()

              
              setInterval() → 一定時間ごとに特定の処理を繰り返す。setInterval(行いたい処理の関数, 繰り返したい時間);

              setTimeout() → 指定した時間後に、1回処理を行う。

              clearInterval() → setInterval() の処理を停止したいときに使う。

setInterval()、setTimeout()の違い

              
              setInterval() → 指定した時間後に必ず関数実行が繰り返される。

              setTimeout() → 関数が実行された後に指定した時間が経過する。

例外処理

              
              toUpperCase() → 文字列をすべて大文字に変える。

              try {} catch {} → エラーが起きても処理を続けたいときに try {} catch {} で囲うと処理を続けられる。
